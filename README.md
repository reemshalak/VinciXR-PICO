VinciXR is a Mixed Reality (MR) drawing assistant built for the **Pico headset** during XR Bootcamp.

It helps users learn how to draw by projecting sketch guides into their real-world space and transforming drawings into interactive 3D models.

Built in **4 days** using only free tools and APIs.
